# MASTER PROMPT: B.O.S (Blueprint of Success) AI Platform

This prompt is designed to instruct an AI system or developer to rebuild or maintain the B.O.S platform.

---

## 🎯 VISION & OBJECTIVE
Build a high-converting, mobile-first AI agent platform for beginner digital entrepreneurs (OFWs). The focus is on **IMMEDIATE ACTION** and **EXTREME SIMPLICITY**.

---

## 🎨 DESIGN SPECIFICATION (AESTHETIC & UI)
- **Theme:** Apple-like Premium Minimalist.
- **Color Palette:** 
  - Background: Deep Black (#000000).
  - Accents: Gold (#D4AF37) used in gradients and headers.
  - Text: White (#FFFFFF) for main content, White/40% for metadata.
- **Typography:**
  - Headlines: "Playfair Display" (Serif) with Italics for a premium look.
  - Body: "Inter" (Sans-serif) for clean readability.
- **Layout:**
  - Single-column focus. No sidebars, no complex navbars.
  - Centered Branding.
  - Hero Section: Massive, direct headline in Tagalog/Taglish.
  - Feature Buttons: Large, interactive cards with clear icons (Lucide-react).

---

## 🧠 USER EXPERIENCE (UX) FLOW
1. **Landed State:** User sees B.O.S logo and 4 main "Gawan mo ako" buttons.
2. **Action State:** Clicking a button replaces the menu with a chat-style textarea.
3. **AI Generation:** Minimal loading state ("B.O.S is thinking...").
4. **Result State:** Content appears in a "Gold Bordered" card with a one-click "Copy" button.
5. **Reset:** Easy "Back" arrow to switch tools.

---

## ⚙️ AI AGENT SYSTEM LOGIC (PROMPTING)
The AI (Gemini 3 Flash) must adhere to these Persona Instructions:

### PERSONA 1: Caption Generator
- **Hook:** Start with a question or a "Saklap" (pain point) factor.
- **Language:** Conversational Tagalog/Taglish (Filipino street-smart tone).
- **Goal:** Emotional connection -> Problem -> B.O.S Solution -> CTA.
- **Optimization:** Optimized for Facebook and TikTok algorithm hooks.

### PERSONA 2: Image Prompt Engine
- **Input:** Short Tagalog descriptions.
- **Conversion:** Expand into highly technical, English-based creative prompts (e.g., photorealistic, 8k, cinematic lighting).
- **Subject matter:** Professional-looking Filipinos, business success, modern lifestyle.

---

## 🛠️ TECHNICAL ARCHITECTURE
- **Framework:** React 19 + Vite (TypeScript).
- **Styling:** Tailwind CSS 4.
- **Animations:** Motion (framer-motion) for "slam-in" transitions and staggered lists.
- **AI Backend:** @google/genai SDK integration (Frontend-Direct).
- **Icons:** Lucide-React.
